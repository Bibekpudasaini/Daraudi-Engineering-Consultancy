# Daraudi Engineering Consultancy Website

This is a responsive multi-page website developed for Daraudi Engineering Consultancy. It represents a real engineering consultancy firm and presents its services, projects, and contact information in a structured and user-friendly format.

## Project Overview

This website was created to demonstrate a professional engineering consultancy website using only frontend technologies. It focuses on clean design, easy navigation, and a simple user experience.

The project includes multiple pages that describe the company profile, services offered, completed projects, and contact information.

## Pages Included

- Home
- About
- Services
- Projects
- Contact

## Technologies Used

- HTML5
- CSS3
- JavaScript (basic functionality)

## Features

- Responsive layout for desktop and mobile devices
- Navigation bar with mobile-friendly menu
- Hero section with project-based image background
- Services section using card-based layout
- Projects section with structured descriptions
- Horizontal scrolling image gallery
- Contact page with form layout and clickable contact links

## Folder Structure

/css        - Contains stylesheet files  
/js         - Contains JavaScript files  
/images     - Contains all project images  
*.html      - All website pages  

## Deployment

This project can be deployed using GitHub Pages:

1. Open repository settings  
2. Go to Pages section  
3. Select branch: main and root directory  
4. Save and wait for deployment  
5. Access the live link generated by GitHub Pages
